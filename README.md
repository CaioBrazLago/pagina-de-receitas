# 🧁 Página de Receitas — Cupcake de Café com Chantilly

Projeto de estudo desenvolvido com **HTML** e **CSS**, criando uma página de receita estática, limpa e bem estruturada.

---

## 📌 Sobre o projeto

Uma página que apresenta a receita de **Cupcake de Café com Chantilly**, com seções de ingredientes e modo de preparo. O foco do projeto está na prática de estruturação semântica do HTML e na aplicação de estilos com CSS puro.

---

## 🎯 Objetivos de aprendizado

- Estruturar conteúdo com HTML semântico (`main`, `section`)
- Aplicar estilos com CSS puro, sem frameworks
- Trabalhar com tipografia, espaçamento e hierarquia visual
- Utilizar fontes externas via **Google Fonts** (Alice)
- Organizar seções de conteúdo com boa legibilidade

---

## 🧩 Seções da página

| Seção | ID | Descrição |
|---|---|---|
| **Sobre** | `#about` | Título da receita, descrição, tempo, rendimento e dificuldade |
| **Ingredientes** | `#ingredients` | Lista com todos os ingredientes necessários |
| **Modo de preparo** | `#preparation` | Passo a passo em parágrafos descritivos |

### 🍰 Receita apresentada

**Cupcake de café com chantilly**
- ⏱️ Tempo: 1h10
- 🍽️ Rendimento: 12 porções
- 💪 Dificuldade: fácil

---

## 🎨 Decisões de estilo

- **Paleta de cores:** fundo em `#f5f4f0` (bege claro), textos em `#1b1b1b` e `#333333`
- **Tipografia:** fonte [Alice](https://fonts.google.com/specimen/Alice) via Google Fonts — serifada e elegante, ideal para receitas
- **Imagem principal:** largura total com `border-radius: 28px` para cantos suaves
- **Layout centralizado:** `max-width: 1120px` com `margin: auto`
- **Hierarquia visual clara:** `h1` (36px) para o título, `h2` (32px) para as seções

## 📚 Conceitos praticados

- Semântica HTML com `main` e `section`
- Reset de estilos com `* { padding: 0; margin: 0; box-sizing: border-box }`
- Centralização de layout com `max-width` e `margin: auto`
- Hierarquia tipográfica com `h1`, `h2` e `p`
- Estilização de listas não ordenadas (`ul`, `li`)
- Integração de fontes externas via Google Fonts
